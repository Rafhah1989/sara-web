# Sara Web

Frontend do sistema SARA (Sistema de Automação de Registros Administrativos), uma aplicação web moderna e elegante projetada para a gestão de produtos, usuários e logística de fretes.

## ✨ Destaques
- **Design Premium**: Interface com temática dourada (Gold System), visual limpo e responsivo.
- **Gestão Completa**: Telas dedicadas para Usuários, Produtos, Setores e Tabelas de Frete.
- **Interatividade Inteligente**: Menu administrativo organizado, busca dinâmica em tempo real e formulários reativos.
- **Validações Robustas**: Máscaras dinâmicas para CPF/CNPJ, Telefone e CEP, com integração automática ao ViaCEP.

## Tecnologias
- Angular 16
- TypeScript
- RxJS (Programação Reativa)
- Vanilla CSS (Design personalizado)
- Angular Router (Navegação SPA)

## Como executar
1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone o repositório e instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm start
   ```
4. Navegue para `http://localhost:4200`.

## Requisitos de Backend
Para pleno funcionamento, esta aplicação requer que o [sara-api](https://github.com/Rafhah1989/sara-api) (ou repositório equivalente) esteja em execução no `http://localhost:8080`.
